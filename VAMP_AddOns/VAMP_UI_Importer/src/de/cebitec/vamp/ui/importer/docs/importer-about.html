About VAMP Import

Reference genome import

Three formats are available for import:

  • Fasta format: Which does not contain any features
  • Genbank format: Can include a vast number of different features
  • EMBL format: Can include a vast number of different features

Track import

Also three formats are available for importing a track:

  • Jok: Output of the short read mapper saruman
  • BAM format: Defined in
  • SAM format: Also defined in

Sequence pair import

VAMP also allows to import Sequence pair data, such as mate pair or paired end data. Requirements: Two mapping files with either all first reads or all second reads of the sequence pair.
