  • EMBL format: Can include a vast number of different features

SAM/BAM Stepwise Parser

This parser is useable for SAM or BAM files. If they are already sorted by read sequence, everything is fine. Otherwise deselect the checkbox "File is sorted by readseq" and VAMP will sort and import the file for you, when clicking "OK".
The step size is the size of a single chunk imported at once in the database. Thus, it should be adapted to the machine the import runs on. For example, a laptop might require a smaller step size. On the other hand, don't choose a too large value for the step size. The import is faster with a medium step size, because the data structures will not grow too large, which means the access to them remains faster.

Sequence pair import

VAMP also allows to import Sequence pair data, such as mate pair or paired end data.
Requirements: Two mapping files with either all first reads or all second reads of the sequence pair.
Distance: Choose the estimated distance from the start of a sequence pair to the end of the pair (e.g. read 1 (100-150), read 2 (300-350) has distance 250).
Direction: The two character code abbreviates the orientation of read 1 and read 2, where "f" means forward and "r" means reverse.
Deviaton: Must be given in percent, e.g. "70" for small pairs which high deviation or "30" for larger mate pairs.
For choosing resonable values it is recommended to analyze the data for these parameters before importing.
