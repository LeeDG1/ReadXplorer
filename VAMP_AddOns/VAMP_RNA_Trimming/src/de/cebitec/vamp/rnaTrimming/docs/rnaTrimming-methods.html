Trimming

With the trimming function a SAM file can be parsed for unmapped reads. In this process the reads will be shortened by using the chosen trimming method.

This component is based on regular exception. There are currently 6 implemented methods, which are available to the user for selection. The following table shows the implemented methods and the result of the given method used on the following example read: AATGCGGTATATAAAAA. The generation of the regular expression used in the program takes place in dependence of maximum length parameter. When using the trimming from both sides (3' and 5') this value is halved, and the appropriate number of nucleotides on each side is shortened. For the same reason, the value range is limited to just even numbers.

Method name Regular expression Example effect
trim poly-A from 3' end (right to left) by variable length ^(.*?)(A{0,4})$ AATGCGGTATATA----
trim poly-A from 5' end (left to right) by variable length ^(A{0,4})(.*?)$ --TGCGGTATATAAAAA
trim poly-A from 3' and from 5' end by variable length ^(A{0,2})(.*?)(A{0,2})$ --TGCGGTATATAAA--
trim all nucleotides from 3' end (right to left) by fixed length ^(.*?)(.{4})$ AATGCGGTATATA----
trim all nucleotides from 5' end (left to right) by fixed length ^(.{4})(.*?)$ ----CGGTATATAAAAA
trim all nucleotides from 3' and from 5' end by fixed length ^(.{2})(.*?)(.{2})$ --TGCGGTATATAAA--

Trim parameters

Reference FASTA file: this reference sequence will be used for the mapping of the trimmed sequences

Mapped SAM Souce File: provide a file which contains some unmapped reads. Only unmapped reads will be extracted in a new FNA file, trimmed and mapped against your reference sequence again.

Mapping parameter: you can provide additional mapping parameters for the used mapping script.

Maximum nucleotides to trim: variable length trim methods will use this length as the maximum number of nucleotides, that will be deleted from the sequence. If you use fixed length trim methods, then this parameter will determine the exact number of trimmed nucleotides. If you use two-sided trim methods, the given number is used as the upper bound for the sum of the trimmed nucleotide count. This means, that only the half of this number is shortened on each side.
