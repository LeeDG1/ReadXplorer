About Thumbnail

The Tumbnail view allows to view the coverage of selected features (e.g. genes) among all opened tracks. Thus you can easily compare the coverage between different experiments (different tracks) and draw conclusions much easier, as you get a direct overview of all opened tracks for the selected feature. Of course you can select and view more than only one feature at once.

Adding features is done by right-clicking the feature and choosing "Add to Thumbnail View".
The thumbnail view can be accessed either by right-clicking an feature or choosing the thumbnail view button in the main icon bar or the "Tools" menu.
