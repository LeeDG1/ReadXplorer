About Transcription Analyses Tool

The transcription analyses module enables you to carry out different automated analyses for transcriptome data sets.

1. The Gene Start Analysis (enabled by checking the "Gene Starts" box) has one default parameter. The minimal increase of the coverage (absolute read counts) from one position to the neighboring one has to be set (default = 50).
Additionally, two more parameters can be set: A maximal initial read count (e.g. 10, default: disabled with 0) can be set and along with this parameter the minimal increase of the coverage for predicted gene starts with less than x (in our example 10) reads at the position directly in front of the gene start has to be set. It can be the same as the mandatory first parameter, but you can also choose a different value, depending on your analysis. If one of these two additional paramters is > 0 the other one also has to be set correctly.

2. The Expressed Genes Analysis allows to flag annotated genes of the reference sequence as expressed genes. Only one parameter needs to be set: The minimum number of reads mapped within the bounds of a gene (default = 1000).

For all these analyses the results are returned in a matrix. Furthermore, they are visualized in the corresponding track viewer by different colors. This helps to compare the different analysis results at a glance.
