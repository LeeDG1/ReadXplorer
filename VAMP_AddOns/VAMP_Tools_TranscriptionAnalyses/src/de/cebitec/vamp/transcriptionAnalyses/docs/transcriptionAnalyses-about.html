About Transcription Analyses

The transcription analyses module enables you to carry out different automated analyses for transcriptome data sets.

1. The Transcription Start Site (TSS) detection (enabled by checking the "TSS detection" box in the wizard) has two default parameters.
A) The minimal number of read starts at the examined position (default = 50 reads starts).
B) The minimal coverage increase in percent from the respective neighboring position to the examined one (default = 50%).

In order to output novel transcript information along with the detected TSS, you can choose the "Novel transcript detection". For detected TSS which do not have neighboring features, it adds the information, that this is a novel transcript and further a transcript stop is inferred by the coverage threshold Minimal transcript extension coverage, which has to be set, if the novel transcript detection checkbox is selected.

Additionally, two more parameters can be set:
C) The maximal low coverage read start number (default = 20 read starts)
D) The minimal low coverage read start number (default = 10 read starts).
For positions with less than x (in the default case 20) read starts at the examined position, but at least y (in the default case 10) read starts, which also fulfill parameter B), a TSS is predicted. It can be the same as the mandatory first parameter, but it is recommended to choose a lower value, depending on your analysis and data quality.

If one of these two additional paramters is > 0 the other one also has to be set correctly.
By checking the Automatic parameter estimation box, the parameters are estimated from the data in the track. In this case the additional parameters are not set, only the mandatory parameters.

2. The RPKM and Read Count Calculation calculates RPKM and raw read count values for annotated features of the reference sequence. Therefore, it requires
A) The minimum number of reads (default = 1 reads)
B) The maximum number of reads (default = 100000000 reads)
mapped within the bounds of an feature.

3. The Operon detection allows to predict operons for prokaryotic genomes on the basis of:
A) Minimal number of spanning reads (default = 5 reads)
between two annotated genes on the same strand. Reads are called "spanning", if they overlap both neighboring features.
For all these analyses the results are returned in a matrix. Furthermore, they are visualized in the corresponding track viewer by different colors. This helps to compare the different analysis results at a glance.
