About the SNP table model

The SNP table model shows the following content:

  1. Pos: The absolute position of the SNP in the reference genome. If it is an insertion it receives a "_(a-z (in ascending order))" as extension.
  2. Track: The name of the track, this SNP is belonging to.
  3. Base: The consensus base found in the mappings at the current position. Meaning the base with the highest coverage.
  4. A/C/G/T/N/_: Number of occurrences of each of these bases or a gap at the current position.
  5. RefCov: Complete coverage of the reference genome by this track at this position. For an insertion the coverage of the position directly before and after is taken into account.
  6. Freq: The frequency of the consensus base among the complete coverage at the current position. For an insertion the coverage of the position directly before and after is taken into account.
  7. Type: Effect type of the SNP. M = Match, S = Substituion, I = Insertion, D = Deletion of the reference base.
  8. Amino SNP: If the current position lies within an feature (e.g. gene) and is not an insertion or deletion the amino acid is calculated for the triplet at the current position incuding the SNP base. The amino acid is represented in one-character-code together with its main chemical property.
  9. Amino Ref: If the current position lies within an feature (e.g. gene) and is not an insertion or deletion the amino acid is calculated for the reference triplet including the current position. The amino acid is represented in one-character-code together with its main chemical property.
  10. Effect on AA: Constitutes the effect of the SNP on the amino acid sequence of its protein. M = match, I = frame shift +1, D = frame shift -1, N = synonymous (neutral) substitution, E = missense substituion.
  11. Feature (Gene): Either the name of the underlying gene, or, if the name is not available, its locus tag. Always shown, if the SNP is located in a gene.
