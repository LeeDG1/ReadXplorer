About VAMP's SNP detection

The snp detection allows to detect snps among all opened/selected tracks for a single reference genome. It opens the dialog windows for creating a snp detection request and returns the result in a new tab, showing all details of the SNPs in a table. The data can be exported into an excel sheet and also used for phylogenetic analysis via fdnaml. For explanations of the data in the table model see SNP table model.

Features of the snp detection:

  • Exon, intron structure is taken into account: If an annotation has subannotations a SNP is only translated, if it is located in an exon. The translation uses the length of the subannotations to determine the correct triplet around a SNP.
  • If an annotation (e.g. gene) has no subannotations, then all SNPs in that annotation are translated with respect to reading frame and strand.
