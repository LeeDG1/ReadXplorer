Transcription Analyses:

The transcription analyses module enables you to carry out different automated analyses for transcriptome data sets: transcription start site detection, operon detection, RPKM value and raw read count calculations. These analyses can be carried out in parallel. After deciding which analyses to run, you have to select the included mapping classes (see Read Classification) and then set the corresponding analysis parameters. The Transcription Analyses toolbar icon (see Main Window a)) becomes available after opening a reference. Alternatively you can start it via "Tools -> Transcription Analyses".
