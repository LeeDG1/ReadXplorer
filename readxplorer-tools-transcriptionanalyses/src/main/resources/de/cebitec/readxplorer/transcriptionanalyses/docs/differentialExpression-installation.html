Differential Gene Expression Analysis Installation Guide

1. Download and install GNU R for your operating system: http://cran.rstudio.com/

2. Launch GNU R with administrator privileges (Windows).

3. Install rJava with the command: install.packages("rJava")

3. R.exe and jri.dll corresponding to your system architecture (x64 or i386, same as used for Java VM) must be added to your PATH variable.

In Windows most of the times:
C:\Program Files\R\R-2.15.3\bin\"architecture"\ and
C:\Program Files\R\R-2.15.3\library\rJava\jri\"architecture"\
The PATH variable can be adjusted at:
Systemsteuerung -> System -> Erweiterte Systemeinstellungen -> Umgebungsvariablen -> Systemvariablen -> Path (in Tabelle)
Both path can be added to the already existing paths contained in the variable. Paths are seperated by ";" like: otherPath;C:\Program Files\R\R-2.15.3\bin\"Architektur"\;C:\Program Files\R\R-2.15.3\library\rJava\jri\"Architektur"\

In Linux:
The following entries must be added to the ".profile" or ".bashrc" file:
export LD_LIBRARY_PATH=/vol/r-2.15/lib/amd64/R/library/rJava/jri:/vol/r-2.15/bin/amd64
export R_HOME=/vol/r-2.15/lib/amd64/R

4. All other packages needed for the analysis are automatically obtained by ftp during the first use of the differential gene expression analysis.
The correct link is already integrated in ReadXplorer, but can be changed under Tools -> Options -> GnuR.
