About VAMP Mapping

The Mapping Module allows to map a sequence read file to a reference sequence. Both files should be in the FASTA format. 

It is possible to configure a mapping script, which gives the full flexibility to process the mapping locally or on a remote machine. 

The script should acceppt the following parameters:

script.sh {reference.fasta} {reads.fasta} {newfiles.basename} {additional.params}

Please see the example mapper scripts:

  • bwa_mapper.sh
  • remote_mapper.sh
