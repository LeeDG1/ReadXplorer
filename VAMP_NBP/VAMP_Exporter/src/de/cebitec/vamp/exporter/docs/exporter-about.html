About VAMP Data Export

All analysis result tables can always be exported into Excel (xls) files. These include an extra sheet with the search parameters and a small statistic for the performed analysis. When the results are too long for one data sheet, another data sheet is added to the file, until all data is stored in the file.

NOTE: For cells with multi-line entries in Excel:
If the line break is not used by default, you have to turn it on:
Mark all data -> right click data -> Format cells -> Alignment -> Check line break -> Click OK
