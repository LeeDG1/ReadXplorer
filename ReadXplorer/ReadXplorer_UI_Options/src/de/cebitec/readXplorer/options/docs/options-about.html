General Options:

Via "Tools -> Options" all available general options can be adjusted.

  • The "General" tab allows setting a proxy.
  • The "Keymap" tab allows adjusting the keyboard shortcuts.
  • The "Miscellaneous" tab allows editing "File", "Output" and "Windows" settings.
  • The "Mapper" tab allows to set the path to an own script, calling a read mapping software (see Mapping). If you want to use the default script supplied with ReadXplorer, do not change this setting.
  • The "Colours" tab allows changing the colors of the three mapping classes (see Read Classification).
  • The "Genetic Code" tab allows to select another genetic code for highlighting of start and stop codons. If the required genetic code is not in the list, it can be added by the user (see screenshot below).
  • The "Gnu R" tab allows editing the CRAN mirror to use for installation of all necessary R packages for the differential gene expression (see Differential Gene Expression) packages DESeq and baySeq. The standard URL leads to the repository of the Bielefeld University. It contains all necessary packages and the available versions have been tested with ReadXplorer and DESeq and baySeq. Therefore, it is highly recommended, to stick to the default CRAN mirror.
  • The "Object Cache" is available for downward compatibility. Older databases (created with versions before ReadXplorer 1.7) with a different structure can be speeded up by using the object cache to cache the coverage of the currently viewed track.
