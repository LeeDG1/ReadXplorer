About ReadXplorer_RNA_Trimming

This module incorporates functions for dealing with rna modifications. Different processes in the cell can lead to a modified transcriptome, that does not match to the refference sequence any more. This is especially polyadenylation.

Typical use case

  1. Mapping of the transcriptome sequence data to a reference genome (use the mapping function)
  2. Import the results data (use the import function)
  3. Trimming of the unmapped transcriptome sequence data and repeat the mapping process with new data (use the trimming function)
  4. Import the new result data (use the import function)
  5. Open the reference sequence and the generated tracks and do manual graphical analysis of coverage
  6. Recognition of similar curve sections by correlation analysis
  7. Detailed observation of these sections in the alignment viewer
