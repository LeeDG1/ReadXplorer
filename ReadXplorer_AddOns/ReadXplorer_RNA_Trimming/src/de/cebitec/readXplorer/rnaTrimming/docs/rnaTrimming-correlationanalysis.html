Correlation analysis

The correlation analysis allows to compare two tracks and find biologically relevant positions with similar coverage curve progression.

Available parameters

Correlation method: choose between PEARSON and SPEARMAN. PEARSON stands for the Pearson product-moment correlation coefficient and should be used if the coverage data could be linearly correlated. SPEARMAN stands for the Spearman's rank correlation coefficient and can be used if the data dependancy is not linear.

Minimum percent of correlation: a correlated interval will appear in the result list only if the chosen minimum correlation is reached.

Minimum peak coverage for both tracks: for each correlated interval the software will determine the peak coverage for both tracks under examination. The minimum of these two values will be compared with the value selected in this setting. If it is less, than the selected value, the interval will not appear in the result list.

Interval length: choose the interval length according to the used read length.
